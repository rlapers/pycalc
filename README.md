Ce projet est une calculatrice très simple qui calcule les opérations d'addition, soustraction, multiplication, 
et division à 2 opérandes maximum.

Pour l'utiliser, il suffit de marquer "python.exe .\pycalc.py [operande1] [operateur] [operande2]" dans le terminal. 
Le programme récupère alors les argumentset fait le calcul demandé.
